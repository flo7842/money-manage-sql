Comment lire un e-mail

Il existe trois façons de lire votre e-mail:

  • dans le client de messagerie
  • dans un navigateur
  • sur papier.

Pour une expérience de messagerie électronique sans culpabilité, nous recommandons les deux premières méthodes.

Lire un courriel avec l'éditeur

Pour lire un e-mail dans l'éditeur:

  1. Cliquez sur l'e-mail pour ouvrir le panneau Message.
  2. Yeux ouverts.
  3. Appliquez les yeux sur l'écran.
    Remarque: si les yeux manquent ou sont déficients, vous pouvez remplacer le logiciel de lecture d'écran.
  4. Lire l'email.
  5. Allez à imprimer.
  6. Sentez-vous coupable à propos des ours polaires.
  7. Annulez l'impression.

Lire un courriel dans un navigateur

Pour lire un e-mail dans un navigateur:

  1. Ouvrez un navigateur.
    Nous recommandons Google Chrome.
  2. Accédez à https://mail.totallyrealdomain.com/.
  3. Cliquez sur l'e-mail pour ouvrir le panneau Message.
  4. Yeux ouverts.
  5. Appliquez les yeux sur l'écran.
    Remarque: si les yeux manquent ou sont déficients, vous pouvez remplacer le logiciel de lecture d'écran.
  6. Lire l'email.
  7. Allez à imprimer.
  8. Sentez-vous coupable à propos des ours polaires.
  9. Annulez l'impression.
